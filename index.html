Features

Everything you need to visualise anything

Built for analysts and developers who want full control without leaving Power BI.

✏️

In-report code editor

A full CodeMirror editor with syntax highlighting, line numbers, search, and replace — inside Power BI's advanced edit mode.

⚡

D3.js v7 built-in

D3 v7 is bundled and exposed globally. No CDN, no version conflicts. Access it via d3 in your scripts.

📊

Live data binding

Your Power BI table data is injected as the pbi.dsv() helper — the same API pattern as D3's CSV loaders.

🎨

8 theme-aware colours

Configurable colour palette in the Format pane, auto-overridden in high-contrast mode for full accessibility.

🔍

JS validation

UglifyJS parses your script before saving. Syntax errors are highlighted and reported with line and column numbers.

♿

Accessible by default

Full keyboard navigation, high-contrast mode support, focus indicators, ARIA labels, and rendering events.

🖱️

Context menu & selection

Right-click context menu, multi-visual cross-filtering, and the Power BI tooltip service — all exposed in pbi.

🌍

Localisation ready

All UI strings are routed through Power BI's localisation manager. Add a .resjson file for your language.

Getting started

Up and running in three steps

No build environment required — just import the visual and start coding.

1

Import the visual

Download the latest .pbiviz from the Releases page, then import it via Insert → Get more visuals → Import a visual from a file in Power BI Desktop or Service.

2

Connect your data

Drag any fields (dimensions, measures, or both) into the Dataset bucket in the Fields pane. Up to 30,000 rows are supported.

3

Open the editor and write code

Click the pencil / Focus mode icon in the visual's top-right corner. The CodeMirror editor opens. Write your D3.js v7 script, then press Save to execute it.

API Reference

The pbi object

Every user script runs with a pbi context object already defined. Use it to access dimensions, data, colours, and Power BI services.

Property / Method Type Description
pbi.width number Usable width of the visual canvas in pixels (after margins).
pbi.height number Usable height of the visual canvas in pixels (after margins).
pbi.colors string[] Array of 8 hex colour strings configured in the Format pane. Auto-substituted with system colours in high-contrast mode.
pbi.isHighContrast boolean true when Windows high-contrast / forced-colours mode is active.
pbi.dsv(accessor?, callback) void Provides report data in a D3-compatible callback. Optionally pass an accessor function to transform rows (same pattern as d3.csv()).
pbi.selectionManager ISelectionManager Power BI selection manager — use for cross-filtering and context menus.
pbi.tooltipService ITooltipService Power BI tooltip service — call .show(), .move(), .hide() to display native tooltips.
pbi.colorPalette IColorPalette Power BI colour palette — access the current theme's colours via .getColor(key).

Global: d3 (v7) is available everywhere in your script. PBI services are also accessible via window.__pbiD3Visual.

Examples

Code samples

Copy-paste these into the editor to get started immediately.

Bar chart

// Simple bar chart using Power BI data
const svg = d3.select("#chart");
svg.attr("width", pbi.width).attr("height", pbi.height);

pbi.dsv(function(data) {
    const x = d3.scaleBand()
        .domain(data.map(d => d.category))
        .range([0, pbi.width])
        .padding(0.2);

    const y = d3.scaleLinear()
        .domain([0, d3.max(data, d => +d.value)])
        .range([pbi.height, 0]);

    svg.selectAll("rect")
        .data(data)
        .join("rect")
        .attr("x",      d => x(d.category))
        .attr("y",      d => y(+d.value))
        .attr("width",  x.bandwidth())
        .attr("height", d => pbi.height - y(+d.value))
        .attr("fill",   pbi.colors[0]);
});

Donut chart

const svg  = d3.select("#chart");
const cx   = pbi.width  / 2;
const cy   = pbi.height / 2;
const r    = Math.min(cx, cy) * 0.9;
const g    = svg.append("g").attr("transform", `translate(${cx},${cy})`);

pbi.dsv(function(data) {
    const pie  = d3.pie().value(d => +d.value);
    const arc  = d3.arc().innerRadius(r * 0.5).outerRadius(r);
    const arcs = pie(data);

    g.selectAll("path")
        .data(arcs)
        .join("path")
        .attr("d",    arc)
        .attr("fill", (_, i) => pbi.colors[i % pbi.colors.length])
        .attr("stroke", "#fff")
        .attr("stroke-width", 2);
});
